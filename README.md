# EuroTour — Туристична агенція (Фінальний проєкт)

Сайт туристичної агенції, орієнтованої на подорожі та незабутні тури країнами Європи. Проєкт виконано в рамках дисципліни «Веб-дизайн та веб-розробка».

## 🔗 Посилання на проєкт
* **Макет у Figma:** [Travel Landing website templates (Варіант: Home-8)](https://www.figma.com/design/0rMwLECwSG82IIahLiE3R1/100--Travel-Landing-website-templates--Community-?node-id=2103-8022&t=0EGwOQuUt8IlnngF-0)
* **Живий сайт (GitHub Pages):** [https://tyshchukmary.github.io/travel-header/](https://tyshchukmary.github.io/travel-header/)

## 🛠 Технологічний стек
* **HTML5** (Семантична розмітка, валідний код)
* **CSS3** (Сучасна верстка за допомогою Flexbox та CSS Grid, плавний скролінг)
* **JavaScript (Vanilla JS)** (Інтерактивність та динамічні ефекти)

## 📌 Реалізовані секції (відповідно до макету)
* **Header** — Головна навігація по сайту
* **Hero Section** — Перший екран із закликом до дії
* **Popular Destinations** — Картки популярних європейських напрямків
* **City Tours** — Блок із міськими турами (банери)
* **Why Choose Tours** — Переваги компанії (іконки та опис)
* **Travel Articles** — Блок новин та статей блогу
* **Footer** — Контакти, швидкі посилання та копірайт

## ⚙️ Функціонал JavaScript
* **Scroll Animation:** Динамічна поява карток, турів та блоків переваг при прокручуванні сторінки (плавний ефект opacity та зміщення).
* **Interactive Navigation:** Активне підсвічування обраних пунктів меню при кліку.
* **Event Handling:** Обробка взаємодії з головною кнопкою замовлення туру (`click event`).

## 💻 Особливості розробки
